ComfyUI Connection
Start ComfyUI with --listen so it accepts network connections:
python main.py --listen 0.0.0.0 --port 8188 --enable-cors-header
If ComfyUI runs on a different machine than SillyTavern, --enable-cors-header is required — without it the browser blocks cross-origin requests even if the port is reachable.

Character LoRAs
Attach a LoRA to the active character. On /imagine the active character's LoRA is injected automatically — switch character and the LoRA follows, no settings change needed. In ComfyUI, title a LoraLoader node IMAGINE_LORA (otherwise the first LoraLoader is used). The list is pulled live from ComfyUI.
